# ふたみ予約システム 実装進捗

## 2026-04-06: ROOMS+PLANS構造移行 & 宿泊料金計算機能

### 実装した機能

1. **ROOMS + PLANS データ構造への移行**
   - 旧FACILITIES配列を、staff.htmlと同じROOMS（物理リソース）+ PLANS（予約プラン）に分離
   - FACILITY_GROUPSを追加（UI表示用のグルーピング）
   - 同じ部屋の宿泊/日帰りがスロットベースで在庫を共有する仕組みを実装

2. **スロットベースの空き判定**
   - isSlotFree(): 部屋+日付+時間スロット単位の空き判定
   - isPlanAvailableForRoom(): プランの全スロットが空いているかを判定
   - getAvailableRooms(): 空いている部屋のリストを返す

3. **予約フロー変更: 施設 -> プラン選択 -> 予約詳細**
   - 施設一覧 -> プラン選択画面（screen-plan-select）-> 予約入力画面
   - 日付検索からもプラン直接選択可能

4. **宿泊プラン: 年齢区分別料金計算**
   - 中学生以上 / 小学生 / 小学生未満の3区分ステッパー
   - 6畳は3名上限制限（超過時に警告表示）
   - リアルタイム料金明細カード（price-breakdown）
     - 基本料金 + 各年齢区分の加算 = 1泊合計
     - 泊数を掛けた総合計
   - 人数・泊数変更のたびに即時更新

5. **日帰りプランはシンプル表示**
   - 年齢区分入力は表示しない
   - 従来通り「1室いくら」の表示

6. **料金体系の正確な反映**
   - 6畳: 基本2,310円/室/泊 + 中学生以上1,580円 + 小学生1,050円 + 小学生未満790円
   - 27畳: 基本9,430円/室/泊 + 同上加算
   - 体験学習室: 基本9,430円/室/泊 + 同上加算
